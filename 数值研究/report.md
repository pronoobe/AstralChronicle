# 五属性 × 八技能回合制RPG平衡结果报告

## 结果概览
这版结果把五个属性分别做成了五种玩法身份：

- 火：爆发输出
- 水：回复 / 续航
- 风：高速连击
- 土：防御 / 反击
- 光：辅助 / 均衡

每个属性都有 8 个技能槽位，其中 7 个主动技能 + 1 个奥义。

## 怎么看表
`skills_40.csv`  
：40 个技能的最终参数。重点看 `kind`、`power`、`cooldown`、`gauge_gain`。

`stage_summary.csv`  
：每个属性在小怪 / 精英 / Boss 三类关卡里的最佳三技能组合表现。

`top_builds_by_stage.csv`  
：每个属性每种关卡下排名前 3 的构筑。

`campaign_summary.csv`  
：把多个子关卡连起来后的线路表现。

`element_summary.csv`  
：把各属性的技能类型分布和关卡表现放在一起。

## 本次平衡的基准
- 小怪：650 HP，85 ATK，35 DEF
- 精英：1300 HP，140 ATK，60 DEF
- Boss：1600 HP，120 ATK，45 DEF，14 回合上限

## 读取结果时的判断
- 小怪胜率接近 100%：说明推进感强
- 精英胜率较高但回合数上升：说明开始需要构筑
- Boss 是否能打穿：看 `win_rate` 和 `avg_turns`
- `campaign_clear_rate`：看整条线是否顺得下来

## 需要注意的点
风属性在这版里被刻意做成高节奏、高波动路线，关卡压力高时会更容易掉线。  
这不是错误，而是它的设计身份之一；如果想让它也成为稳定通关型职业，可以继续把 `疾风刺`、`旋羽连射` 和 `天穹飓风` 再抬一点。

## 本次结果的一个直观结论
优化并没有把所有属性都压成同一套答案，而是保留了不同关卡下的多种优解：
- 火 / 土 / 光 更偏稳健
- 水 更偏续航
- 风 更偏高速和吃节奏
